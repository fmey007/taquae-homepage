# Taquae - Homepage

Homepage du cabinet de conseil en stratégie IA pour PME et startups.

## À propos

Taquae est un cabinet de conseil spécialisé en audit, accompagnement et mise en œuvre stratégique de l'IA pour les PME, startups et organismes publics.

## Contenu

- `Homepage.html` — Maquette interactive de la homepage
- `assets/` — Logo et ressources visuelles
- `uploads/` — Images générées pour la maquette

## Pour démarrer

1. Ouvrez `Homepage.html` dans votre navigateur pour voir la maquette interactive
2. Le code source HTML/CSS/JS est directement dans le fichier pour prototype rapide
3. Pour la mise en production, récréez la structure en React/Vue ou framework de votre choix
